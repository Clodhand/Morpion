# Morpion

Projet Python autour du jeu du morpion.

## À propos

Ce dépôt regroupe plusieurs essais réalisés pour pratiquer Python à partir d’un jeu simple.  
L’idée était de travailler à la fois la logique du programme, l’organisation du code et quelques aspects plus visuels avec une interface graphique.

## Ce que j’y ai appris

- manipuler une grille et des états de jeu
- gérer les tours des joueurs
- détecter les victoires et les matchs nuls
- structurer un petit projet Python
- expérimenter avec la programmation orientée objet

## Contenu du dépôt

On y trouve différentes versions ou essais autour du morpion, selon mon avancement.
